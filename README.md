# Titulo del Repositorio

Prueba Tecnica JDigital

## Para empezar

descargar el repositorio, usando git clone



### Instalacion

Paso 01 Intalar dependencias del backend en la carpeta raiz

```
npm i
```

Paso 02 Intalar dependencias del frontend en la carpeta client

```
npm i
```

Paso 03 Intalar para arrancar la aplicacion en cada terminal ya sea del backend y del frontendusar:

```
npm run dev
```

## Plataforma

* **Para la aplicacion frontend se uso Vite y para el backend las librerias que se describen posteriormente **
